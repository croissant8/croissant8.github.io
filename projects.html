Проекты

Финансовый аутсорсинг Данный статический сайт-визитка разработан по заказу аутсорсинговой компании. Благодаря разработке сайта с учетом требований SEO, сайт продолжительное время держался на 11-20 позициях в результатах выдачи поисковых систем Google, Yandex без дополнительного продвижения.
Учебный проект Данный сайт разработан на основе индивидуального учебного задания. Требовалось организовать навигацию, использовать фреймы для трехоконной структуры, использовать бегущую строку в "шапке", разный шрифт, заливку страниц, вставить списки. С помощью программы GeoHTML сформировать навигационную карту.
